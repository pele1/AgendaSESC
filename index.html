AgendaSESC

Autor: Steffen Retzlaff

AgendaSESC é um aplicativo Android para mostrar a programação e cursos do SESC São Paulo.
Android app which shows the content "Programação" and "Cursos" from http://www.sescsp.org.br/

O projeto é um desenvolvimento independente e particular.

Para instalar o app é preciso baixar o arquivo AgendaSESC.apk: https://github.com/pele1/AgendaSESC/blob/master/bin/AgendaSESC.apk

Até o momento o aplicativo não está disponivel no PlayStore.
